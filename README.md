## Quick Installation

    git clone https://github.com/mitulrp-github/Laravel_IMS.git

    cd Project/
    
### Composer

    composer update

### Create Postgres Database Schemas Name "IMS"   
    DATABASE=postgres
    USERNAME=postgres
    
### For Environment Variable Create
 
    cp .env.example .env
 
    
 ### For Migration table in database [Create database name as ```IMS```]
 
    php artisan migrate
    
### Server ON ```url: http://127.0.0.1:8000/```

    php artisan serve
